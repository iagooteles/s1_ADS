4. Análise de dados

Mcfedries (2020) define análise de dados como a aplicação de ferramentas e técnicas para organizar, estudar, chegar a conclusões e, às vezes, fazer previsões sobre uma coleção específica de informações. Portanto, a análise de dados é uma etapa dos processos de pesquisa que ocorre logo após a coleta de dados, transformando dados brutos em informações organizadas que podem ser interpretadas e avaliadas. Esse processo permite alcançar novos conhecimentos científicos. Por esse motivo, a etapa de análise de dados é uma das mais importante no processo da investigação científica.
